Guía de estilos

Colores corporativos

Principalmente usamos nuestros dos colores corporativos y el color acentuado. Siempre respetando el ratio de contraste óptimo entre texto y background.

Colores primarios

Primary RED recomendado para backgrounds, CTA, textos destacados, etc.

Secondary ORANGE usado en botones, elementos de interfaz y en nuestra línea gráfica.

Accent BLUE se debe usar con moderación, para CTA, elementos destacados, etc.
